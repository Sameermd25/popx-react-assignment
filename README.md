## 📖 Project Overview

PopX is a modern user onboarding application that allows users to:

* Create a new account
* Log in with registered credentials
* View profile information
* Navigate smoothly between screens
* Persist user data using localStorage

The application is designed to closely match the provided UI mockups while maintaining clean code architecture and reusable components.

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone <your-repository-url>
```

### Navigate to Project Directory

```bash
cd popx-react-app
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```bash
http://localhost:5173
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview Production Build:

